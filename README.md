# CitrusOS
CitrusOS is a custom subsystem for Windows that is based of Debian Linux. You can download the opt/citrus/bin files to download the custom commands.

If you would like the custom prompt, replace your current ~/.bashrc with the code I've supplied

## Big Bugs

- Citrusfiles cannot open certain types of files
- Listusers does not have any authority checks - Only use this for development if you trus other users on your desktop
  
## Features

- Custom Citrus shell
- Linux-style commands
- citrusedit text editor
- Citrus Explorer
- Custom tools and utilities

## Installation

Download the files I've supplied, and place them in /opt/citrus/bin (create the directory if you haven't already)

Edit your ~/.bashrc and replace the code with the code supplied. Hope you enjoy it!
